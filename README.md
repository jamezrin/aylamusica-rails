# Aylamusica

## Comandos

- Lanzar la aplicación

    ```bash
    rails server
    ```

- Crear canción manualmente

    ```bash
    rake app:create_song
    ```
  
- Volver a migrar desde cero
    
    ```bash
    rake db:migrate:redo
    ```
